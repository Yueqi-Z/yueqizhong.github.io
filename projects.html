From Java Development to Data Visualization

01. Java Red Packet Rain Lottery System

A high-concurrency lottery system designed to handle massive user participation in real time. This system enables users to grab digital red packets during specific time intervals, ensuring efficient prize distribution and seamless user experience under heavy traffic conditions. Key features include robust prize management, real-time interaction, and scalability.

02. Java Carpool System

A web-based application that connects drivers and passengers for shared rides. This project focuses on trip matching, real-time trip status updates, and seamless payment processing.

Integrated with AI-based license plate recognition, the system ensures smooth communication between drivers and riders.

03. Tableau Dashboard for UW Transportation

An interactive data visualization dashboard developed for UW Transportation to examine parking patterns. By utilizing historical data alongside weather analyses, this dashboard offers valuable insights into parking occupancy, turnover rates, and trends, aiding in more effective resource management and planning.
